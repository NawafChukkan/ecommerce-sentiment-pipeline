Pipeline Flow

Source generation, processing, storage, and follow-up analysis path.

1. Seed Data

Synthetic products, orders, reviews, and inventory snapshots are generated at scale.

2. Raw Zone

The source files are stored under the raw zone and can be routed through a blob-storage profile.

3. Spark Processing

PySpark calculates trend, lifecycle, category, and risk metrics.

4. SQL Marts

Processed outputs are written to PostgreSQL for dashboard and follow-up analysis use.

5. Dashboard

The app provides interpretation, filtering, narrative support, and explorer views.
